GhostFiveM

Projeto C++ para Windows com interface e funcionalidades internas.

O repositório principal contém código nativo para Visual Studio, não uma aplicação web.

Como usar

  1. Abra GhostFivem.sln no Visual Studio.
  2. Compile em x64 e Release.
  3. O binário será gerado em src\x64\Release.

Deploy no Vercel

Use este repositório no Vercel como um site estático de apresentação. O código em src/ não roda no Vercel.
